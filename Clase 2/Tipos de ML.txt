Tipos de Machine Learning
Existen diferentes tipos de técnicas o algoritmos de aprendizaje de maquina; dentro de ellos encontramos los algoritmos supervisados, semi-supervisados, no supervisados y de aprendizaje reforzado, profundizando en cada uno de ellos podemos encontrar que los algoritmos supervisados son aquellos en donde se introduce en el dataset de entrenamiento la variable respuesta para que el algoritmo calcule la mejor función para predecirla basad en las variables explicativas, en este se encuentran las técnicas de regresión y modelos predictivos, también existen los algoritmos semi-supervisados en donde se encuentra la variable respuesta para una parte de los datos y para otra parte no,  en el aprendizaje no supervisado no existe una variable a explicar o predecir, por lo tanto los modelos son ajustados con las variables que se cuente, estos usualmente se usan para identificación de patrones o clusterización, finalmente el aprendizaje reforzado es el modelo que se reentrena así mismo con las nuevas observaciones buscando maximizar o minimizar un valor en función de una meta (función objetivo)
